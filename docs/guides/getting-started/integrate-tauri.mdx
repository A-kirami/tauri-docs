# Integrate Tauri into your existing project

This guide will walk you through integrating tauri into your existing project.

:::info

Before we continue, make sure you have completed the [prerequisites] to have a working development environment.

:::

## Intro

If you are here that means you already have a web project whether it is Node.js-based,
like [Svelte], [React], [Vue] or even [Angular] or Rust-based like [Yew], [Dominator]
and you are looking to integrate tauri into it.

For the purposes of this guide we will use a [React] project that has been created using [create-react-app] and has this structure:

```

```

## Integrate Tauri

### 1. Add Tauri CLI

To integrate Tauri into your project, you first need to install the Tauri CLI as a dev dependency or globally if you prefer.

```shell
yarn add -D @tauri-apps/cli
```

:::info

You don't have to use [Yarn], you can use the npm package manager you use with your existing project or use [Cargo] if you have a rust project.

:::

### 2. Intialize Tauri project

Now you just can initialize tauri by running:

```shell
yarn tauri init
```

### 3. (Optional) Add JS api package

You can also add The Javascript api package to use Tauri's [APIs]

```shell
yarn add @tauri-apps/api
```

## Invoke Commands

<Commands />


```jsx title=src/App.js {4,7-12}
import logo from './logo.svg';
import './App.css';

import { invoke } from '@tauri-apps/api'

function App() {
  // now we can call our Command!
  // Right-click the application background and open the developer tools.
  // You will see "Hello, World!" printed in the console!
  invoke('greet', { name: 'World' })
  // `invoke` returns a Promise
  .then((response) => console.log(response))

  return (
    // -- snip --
  )
}
```

:::tip

If you want to know more about the communication between Rust and JavaScript, please read the Tauri [Inter-Process Communication][inter-process-communication] guide.

:::
